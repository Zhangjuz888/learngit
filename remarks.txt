title: 创建本地git仓库
author: zhangjuz
time: 2019-01-10
步骤:
1、cd  // 切换更目录
2、创建文件目录mkdir "目录名"
3、vi "文件名" 并打开该创建的文件
4、pwd 查看当前文件路径
5、git init 把在本地新创建的目录改为可管理的git仓库

title: 回退指定版本
author: zhangjuz
time: 2019-01-10
步骤:
1、查看版本号 git log
2、git reflog 查看Git记录的指令记录
3、git reset --hard "版本号"

title: git 工作流程
author: zhangjuz
time: 2019-01-10
工作区: 本地目录文件夹存放文件的地方
暂存区/缓存区: git add 缓存修改文件的地方
版本库: git commit 所提交所在分支的地方
远程库: 将版本库里面内容push 到 远程仓库存放的地方
